## Цели работы

1. Изучить принципы объектно‑ориентированного программирования на Python.  
2. Научиться проектировать классы с использованием геттеров и сеттеров (`@property`).  
3. Реализовать базовые операции над коллекцией объектов (добавление, поиск, сортировка).  
4. Изучить сохранение и загрузку данных в формате CSV и Pickle.

## Структура модулей

- **book.py**  
  - Класс `Book` с приватными атрибутами `_title`, `_author`, `_year`.  
  - Свойства `title`, `author`, `year` через декораторы `@property` и валидация года в сеттере.  
  - Магический метод `__str__` для форматированного вывода информации о книге.

- **library_catalog.py**  
  - Класс `LibraryCatalog` для управления списком книг.  
  - Методы:  
    - `add_book(book)` — добавление объекта `Book`.  
    - `find_books_by_author(author)` — поиск по автору (без учёта регистра).  
    - `sort_books_by_year()` — сортировка списка книг по году издания.  
    - `save_to_csv(filename)` / `load_from_csv(filename)` — сохранение и загрузка списка в/из CSV-файла.  
    - `save_to_pickle(filename)` / `load_from_pickle(filename)` — сериализация/десериализация каталога через `pickle`.

- **main.py**  
  - Интерфейс командной строки с меню:  
    1. Добавить книгу  
    2. Найти книги по автору  
    3. Отсортировать книги по году  
    4. Сохранить в CSV  
    5. Загрузить из CSV  
    6. Сохранить в pickle  
    7. Загрузить из pickle  
    8. Выход  
  - Обработка ошибок через `try/except`, повтор запросов для корректного ввода.

## Описание ключевых решений

1. **Геттеры и сеттеры**  
   Свойства `title` и `author` простые, а в сеттер `year` добавлена проверка: 0 ≤ year ≤ 2025, иначе `ValueError`.

2. **Магические методы**  
   Реализован `__str__` в `Book` для удобного вывода объекта как строки.

3. **Работа с файлами**  
   - CSV: стандартный модуль `csv`.  
   - Pickle: модуль `pickle` для сериализации всего объекта `LibraryCatalog`.

4. **Обработка исключений**  
   - Некорректный ввод года в цикле `while` переспрашивается.  
   - При загрузке CSV/Pickle ловятся `FileNotFoundError` и `ValueError`.
